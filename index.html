ESTA GUÍA VA DIRIGIDA A LOS PARTICIPANTES DE LA ESCUELA DE PRIMAVERA DE LA SEES 2026

Los siguientes pasos son con la finalidad de que los participantes instalen de manera previa las herramientas libres (Librelane) para ejecutar el flujo de diseño de un ASIC. Se entiende que todos los participantes están usando Windows 10+ como sistema operativo (OS) en sus computadoras personales. En caso de tener un OS diferente por favor contactar con los organizadores de la Escuela (sgarduno@cinvestav.mx, jsanchezl@cinvestav.mx).

Pasos que seguir:

1. Abrir PowerShell en modo administrador (botón de inicio + buscar (PowerShell) + clic derecho sobre el ícono + ejecutar como administrador).
2. Instala WSL en tu PC, escribe y ejecuta (dar Enter):
wsl --install
3. Si no funciona revisa el siguiente enlace: https://docs.microsoft.com/en-us/windows/wsl/install. Si funcionó solo salta al siguiente paso.
4. Reinicia tu computadora.
5. Abrir PowerShell en modo administrador (ver paso 1).
6. Instala Ubuntu escribiendo y ejecutando (dar Enter):
wsl --install -d Ubuntu
7. Revisa que se instaló correctamente con:
wsl --list --verbose
Deberías ver en consola algo como lo siguiente:
PS C:\Users\user> wsl --list --verbose NAME STATE VERSION * Ubuntu Running 2
8. Ve al menú de inicio (botón de inicio) y busca Ubuntu y da clic sobre su ícono. Esto abrirá la terminal de Ubuntu/Linux en tu PC de Windows.
9. Escribe y ejecuta (Enter) el siguiente comando:
sudo apt-get install -y curl
10. Escribe y ejecuta (Enter) el siguiente comando:
curl --proto '=https' --tlsv1.2 -fsSL https://artifacts.nixos.org/nix-installer | sh -s -- install --no-confirm --extra-conf " extra-substituters = https://nix-cache.fossi-foundation.org extra-trusted-public-keys = nix-cache.fossi-foundation.org:3+K59iFwXqKsL7BNu6Guy0v+uTlwsxYQxjspXzqLYQs= "
11. Seguidamente escribe y ejecuta (Enter) este comando:
. /nix/var/nix/profiles/default/etc/profile.d/nix-daemon.sh
12. Asegúrate de cerrar la terminal de Ubuntu una vez finalizado el paso anterior y ábrela nuevamente (ver paso 8).
13. Clonar Librelane con este comando:
git clone https://github.com/librelane/librelane
14. Aparecerá una carpeta llamada librelane.
15. Puedes verla escribiendo y ejecutando (dar Enter) en la terminal de Ubuntu:
ls
16. Entra a la carpeta librelane escribiendo y ejecutando (Enter):
cd librelane/
17. Una vez dentro de la carpeta librelane, ejecuta (escribe y da Enter):
nix-shell
18. Ejecuta el siguiente comando (escribe y da Enter):
librelane --log-level ERROR --condensed --show-progress-bar --smoke-test
19. Listo ya tienes instalada las herramientas libres para correr todo el flujo de diseño de un ASIC, ya tienes todo listo para la Escuela de Primavera. ¡¡FELICIDADES!!
